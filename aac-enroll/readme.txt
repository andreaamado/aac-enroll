== Description ==
Create a medical appointment form where the patient can send message to specific doctors, these doctors are WP users.

= How to Use =
Activate the plugin, create one page where you wanna show the form and add the shortcode [aac_enroll]

= Tips =
This plugin generate a page called “Form – Enroll Plugin”, that's necessary to show the form using ajax. If for some reason you edited this page the plugin can stop working well, in this case you just have to deactivate and activate the plugin again.
